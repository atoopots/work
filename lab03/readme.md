# Lab03 ESP32 PWM呼吸灯实验

## 实验目标

使用ESP32板载GPIO2 LED，通过PWM脉冲宽度调制实现平滑呼吸灯光效果。

## 实验器材

ESP32开发板、USB-C数据线、安装Arduino IDE的电脑

## PWM原理

PWM脉冲宽度调制依靠快速开关数字信号，通过调整占空比改变LED平均亮度；8位分辨率亮度范围0\~255，数值越大亮度越高。

## 实验操作步骤

1. ESP32通过USB-C数据线连接电脑，确认电源指示灯点亮；
2. Arduino IDE配置开发板为ESP32 Dev Module，选择对应COM串口；
3. 烧录lab03\_breath.ino程序，上传失败时按住BOOT按键重试；
4. 打开波特率115200串口监视器，每轮呼吸完成会打印日志；
5. 修改delay()参数调整呼吸快慢，数值越大呼吸越缓慢。

## 常见问题排查

1. LED不亮：核对开发板型号、串口、引脚，可更换GPIO13测试；
2. 渐变效果不明显：调高resolution分辨率、增大delay延时；
3. 程序上传失败：上传时按住BOOT，更换支持数据传输的USB线。

## 核心知识点

ledcAttach()：绑定引脚并配置PWM频率、分辨率；
ledcWrite()：输出指定占空比控制LED亮度；
双层for循环递增/递减亮度值，实现明暗平滑渐变呼吸效果。
